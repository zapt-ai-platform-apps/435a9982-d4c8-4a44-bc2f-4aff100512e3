# Gear Customization Journey

This journey explains how to upgrade and customize your sniper gear.

## Steps

1. **Accessing Customization**  
   From the Home Menu, click the "Customize Gear" button to navigate to the Gear Customization screen.

2. **Editing Gear Options**  
   The screen provides input fields for gear details such as Scope Type and Bullet Type. Default values are pre-filled.

3. **Saving Changes**  
   After updating the gear settings, click the "Save & Return" button. A loading state is shown during saving, and your choices are logged.

4. **Navigation**  
   You can also cancel the changes by clicking the "Cancel" button, which returns you to the Home Menu.
